# SQL Learning & Practice Repository

This repository documents my **SQL learning journey**, covering fundamental to advanced database concepts with structured examples and practice queries.

The goal of this repository is to build a strong foundation in SQL for **Data Analytics, Data Engineering, and Database Management**.

---

## Repository Overview

This repository contains SQL scripts organized topic-wise to make learning and revising SQL concepts easier.

Each folder includes SQL queries related to a specific concept such as joins, aggregations, subqueries, and window functions.
